Google OSINT diggers rejoice!
Whether you are looking for cardable sites, trying to find exposed plaintext password files, searching for particular content
Whatever the need! Dorky Malorky is the convenient Chrome Browser Extension you need

TO USE:
extract the folder onto your device
go to chrome
click three dots at top right
go to < EXTENSION < MANAGE EXTENTIONS
Make sure Developer Mode is enabled at the top right
CLICK LOAD UNPACKED
Select the extracted folder
Hit a Select folder
Turn on extension and PIN for easy access


Click extension and click any Google Dork search query you so desire and it will
automatically search google for that google dork


Happy Digging.
